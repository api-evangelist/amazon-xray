# Amazon X-Ray

AWS X-Ray is a distributed tracing service that helps developers analyze and debug production applications, providing end-to-end visibility into requests as they travel through the application.

## APIs

- **Amazon X-Ray REST API** - RESTful API for AWS X-Ray distributed tracing operations including trace management, service maps, sampling rules, groups, and insights for application performance analysis.
